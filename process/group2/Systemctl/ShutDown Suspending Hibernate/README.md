## ปิดระบบปฏิบัติการ
```
systemctl poweroff
```
> โดยปกติ, การรันคำสั่งเหล่านี้จะทำให้ systemd ส่งข้อความแจ้งเตือนถึงผู้ใช้ทุกคนที่เข้าสู่ระบบ. หากต้องการป้องกัน systemd จากการส่งข้อความนี้, ให้รันคำสั่งนี้พร้อมกับตัวเลือก --no-wall

```
systemctl --no-wall poweroff
```
> เพื่อปิดระบบปฏิบัติการโดยไม่ตัดกำลัง

```
systemctl halt
```

## รีสตาร์ทระบบปฏิบัติการ
```
systemctl reboot
```
> โดยปกติ, การรันคำสั่งเหล่านี้จะทำให้ systemd ส่งข้อความแจ้งเตือนถึงผู้ใช้ทุกคนที่เข้าสู่ระบบ. หากต้องการป้องกัน systemd จากการส่งข้อความนี้, ให้รันคำสั่งนี้พร้อมกับตัวเลือก --no-wall

```
systemctl --no-wall reboot
```


## หยุดระบบปฏิบัติการ
```
systemctl suspend
```

## หลับระบบปฏิบัติการ
```
systemctl hibernate
```
> เพื่อหยุดและหลับระบบปฏิบัติการ

```
systemctl hybrid-sleep
```
